Update page templates

Each of the four types of update pages is built for a specific type of article, or publication. They are visually very similar, with small differences in their components customized to the type of publications needs.

  • Press release page template
  • Record page template
  • Tips for treasurers page template
  • Weekly digest page template

Components

Shared across all four templates

  • Page type tag
    This module is applied automatically when you select the page template
  • Title
  • Date
  • Category
  • Body
    A paragraph is the standard option, but the body module can include:
    • Paragraph
    • HTML
    • Image
    • Table
    • Custom table
  • Authors
  • Read next
    This module auto-fills with the last article of the same type published before the one being edited, unless the author changes it

Unique to Record page templates

  • Monthly issue
  • Tags
  • Related section

When to use

Each publication should use its corresponding specific page template.
