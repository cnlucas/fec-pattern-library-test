Reporting example page template

Reporting example pages are Wagtail page templates. They are used specifically for “How to report” content that supports specific guide topics, or for Example scenario pages (which have not been put into use yet, but share the design intentions in the template).

Components

  • Title
  • Pre-title
    “How to report,” “Example scenario”
  • Feature image
    Used only for example scenario pages. If included, it will be cropped to a small circle and appear to the left of the page title.
  • Body
    The body module can include:
    • Paragraph
    • Example image
    • Reporting example cards
  • Related media

When to use

  • Pages about "How to report"
  • Example scenario pages for a fictional in-depth case study

Example pages

  • How to report individual contributions
  • How to report joint contributions
  • How to report in-kind contributions
