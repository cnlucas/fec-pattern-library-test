Custom page template

The custom page template is the most basic-level article page about a single, narrow topic. It has the most flexibility in embedded modules to accommodate the many scenarios it is used for.

Components

  • Title
  • Body
    A paragraph is the standard option, but the body module can include:
    • Paragraph
    • HTML
    • Image
    • Table
    • Example paragraph
    • Example forms
    • Reporting example cards
  • Sidebar, which can contain:
    • Heading
      The title of the sidebar
    • Paragraph
    • HTML
    • Record articles
      Appears below the sidebar
    • Contact link
      Appears below the sidebar
  • Continue learning
  • Citations (legal citations)

When to use

  • Guide pages
  • Data file description and metadata pages
  • Pages about a single, narrow topic
